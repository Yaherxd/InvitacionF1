Conclusión

Durante la realización de esta práctica, aprendí a estructurar una página web utilizando únicamente HTML y CSS. Comprendí la importancia de las etiquetas para organizar correctamente el contenido y mejorar la accesibilidad. También exploré cómo los estilos en CSS pueden cambiar el aspecto de los elementos para hacerlos más atractivos e interactivos.

Aprendí a usar enlaces para conectar diferentes páginas dentro de un sitio web, así como a insertar imágenes utilizando rutas locales. Además, implementé efectos visuales con CSS para mostrar imágenes al pasar el cursor sobre un botón, sin necesidad de JavaScript, lo que mejora la experiencia del usuario de forma eficiente y sencilla.

La aplicación de estos conocimientos es muy útil para desarrollar páginas informativas, invitaciones interactivas, portafolios y sitios web básicos sin requerir programación avanzada. Gracias a esta práctica, puedo seguir mejorando en el diseño web y aplicar estos conceptos en futuros proyectos.
